PASTAS QUE DEVEM CONTER PARA O FUNCIONAMENTO:

ANSW - TODAS AS RESPOSTAS DAS REQUISIÇÕES (.ANS)

FILES - QUALQUER ARQUIVO

REQS - REQUISIÇÃO NA PASTA FILES NO DIRETÓRIO DO PROGRAMA
	   .::: IMPORTANTE: O FORMATO DA REQUISIÇÃO DEVE SEGUIR
	   					O PADRÃO ABAIXO.

	   					NOME: req<número da requisição>.req

	   					CONTEUDO: GET <nomedoarquivo.extensão>

SOURCE-CODE - CODIGO FONTE

OBS: TODOS OS DIRETÓRIOS DESCRITOS ACIMA PODEM SER 
	 MUDADOS MODIFICANDO OS DADOS NA CLASSE RUNNABLE

====================================================
((((((((((((((((((  COMPILING  )))))))))))))))))))))
====================================================

1. ABRIR NO TERMINAL O CÓDIGO FONTE DO PROJETO, LOCA-
   LIZADO NA PASTA [SOURCE-CODE].

2. DIGITE O COMANDO: make clean

3. EM SEGUIDA DIGITE: make 

4. O PROGRAMA INICIALIZARA ESPERANDO O USUÁRIO DIGITE
   AS REQUISIÇÕES DESEJADAS, SEGUINDO PADRÃO ABAIXO:

   INSTRUÇÃO: all  	FUNÇÃO: CARREGA TODAS REQUISIÇÕES
   INSTRUÇÃO: numeros <space>  FUNÇÃO: DEFINIÇÃO DE REQUISIÇÕES
	
..:: IMPORTANTE: PARA ESCOLHER AS REQUISIÇÕES DESEJADAS É
	 		     NECESSÁRIO O NÚMERO REFERENTE A REQUISIÇÃO
				 SEPARADO POR ESPAÇO INCLUSIVE O ULTIMO NÚMERO.
